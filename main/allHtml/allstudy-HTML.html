니코 라칸 나미

table 코드 설명:

  1. <table> (속성) </table>: table태그를 먼저 선언합니다.

  2. <tr>:(table row) 테이블의 한 행을 나타냄.

  3. <th>:(table header) 제목 셀 (굵고 표시 되고 가운데 정렬)

  4. <td>:(table data) 테이블의 일반 데이터 셀.
